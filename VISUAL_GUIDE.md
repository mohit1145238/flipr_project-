## âœ¨ Animation Effects

### Scroll Animations (AOS)
- **fade-down**: Elements fade and slide down
- **fade-up**: Elements fade and slide up
- **fade-up-left**: Slides up from left
- **fade-up-right**: Slides up from right
- **zoom-in**: Zoom and fade effect

### Hover Animations
- **Cards**: Lift up, border color change, shadow increase
- **Buttons**: Scale down briefly, shadow increases
- **Images**: Scale up slightly, filters apply

- **Floating Circles**: Smooth up/down movement
- **Gradient Shifts**: Subtle color transitions
- **Glow Effects**: Pulsing shadow on hover

### Interactive Animations
- **Form Focus**: Blue glow on input fields
- **Error State**: Red border and red glow
- **Success**: Notification slide in/out
